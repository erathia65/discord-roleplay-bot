# Source Directory

Contains the main bot implementation.

## Files

- `bot.js` - Main bot logic and Discord interaction handling
- `GameEngine.js` - Core game logic and state management
- `StoryManager.js` - Story hooks and narrative generation
- `ChoiceHandler.js` - Button interaction processing
- `Database.js` - Game state and history persistence (optional)

## Usage

Import modules:
```javascript
const GameEngine = require('./src/GameEngine');
const DiscordRoleplayBot = require('./src/bot');
```

## Development

Add new features in appropriate module files. Ensure all functions are documented with JSDoc comments.
