# Templates Directory

Story templates and narrative scenarios.

## Files

- `fantasy.js` - Fantasy story scenarios
- `scifi.js` - Science fiction scenarios
- `horror.js` - Horror scenarios
- `adventure.js` - Adventure scenarios

## Usage

```javascript
const templates = require('./templates/fantasy');
const storyHook = templates.getRandomHook();
```

## Adding Templates

Create new template files following the same structure as existing ones.
